## Description

This assignment covers several topics from discrete mathematics including **set theory, asymptotic analysis, and number theory**.
The problems involve mathematical reasoning and formal proofs.

## Topics Covered

* Set operations and properties
* Functions and mappings
* Asymptotic notation (Big-O)
* Mathematical proofs
* Divisibility and number theory

## Problem Overview

### Question 1 – Sets and Functions

Problems about properties of set operations, injective functions, power sets, and reasoning about mappings between sets.

### Question 2 – Algorithm Analysis

Questions related to asymptotic growth and Big-O notation.

### Question 3 – Divisibility and Number Theory

Proof-based questions involving rational numbers, prime numbers, and properties of divisibility.
