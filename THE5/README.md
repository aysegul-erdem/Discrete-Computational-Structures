# Take Home Exam 5

This folder contains my solution for **Take Home Exam 5** of the course
**CENG223 – Discrete Computational Structures** at **Middle East Technical University (METU)**.

## Description

This assignment focuses on concepts from **graph theory** and related algorithms.
The problems involve analyzing properties of graphs, reasoning about graph structures, and applying classical graph algorithms.

## Topics Covered

* Euler paths and circuits
* Hamilton paths and circuits
* Graph isomorphism
* Bipartite and regular graphs
* Graph algorithms (Prim, Kruskal, Dijkstra)

## Problem Overview

### Question 1 – Euler and Hamilton Properties

Analyze a given graph to determine whether it contains Euler paths, Euler circuits, Hamilton paths, or Hamilton circuits.

### Question 2 – Graph Isomorphism

Determine whether two graphs are structurally equivalent (isomorphic) and justify the answer.

### Question 3 – Bipartite and Regular Graphs

Work with definitions and properties of bipartite and regular graphs, including proof-based reasoning.

### Question 4 – Graph Algorithms

Apply classical graph algorithms to a weighted graph, including minimum spanning tree algorithms and shortest path computation.
