# Take Home Exam 4

This folder contains my solution for **Take Home Exam 4** of the course
**CENG223 – Discrete Computational Structures** at **Middle East Technical University (METU)**.

## Description

This assignment focuses on topics from discrete mathematics such as recurrence relations, generating functions, and properties of relations.
The problems involve mathematical analysis, proof techniques, and reasoning about relational structures.

## Topics Covered

* Recurrence relations
* Characteristic equations
* Generating functions
* Equivalence relations and partial orders
* Hasse diagrams and lattice structures

## Problem Overview

### Question 1 – Recurrence Relations

Solve a recurrence relation using both the characteristic equation method and generating functions.

### Question 2 – Recurrence Transformation

Analyze a recurrence relation by transforming the sequence into a different form.

### Question 3 – Generating Functions

Find the generating function of a given sequence and express it in closed form.

### Question 4 – Properties of Relations

Determine whether several statements about relations are true or false and justify the answers with proofs or counterexamples.

### Question 5 – Divisibility Relations

Work with a divisibility relation on a set, draw its Hasse diagram, and analyze how the structure changes when modifying the set.
